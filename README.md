# 🎬 Compresor de Video — Browser

Reduce el tamaño de tus videos directamente en el navegador.
Sin servidores. Sin subir archivos. 100% privado.

## ✨ Características
- Powered by **FFmpeg.wasm** (corre en el navegador)
- Control de calidad con CRF (18 = alta calidad, 35 = máxima compresión)
- Cambio de resolución: Original / 720p / 480p / 360p
- Control de bitrate de audio o eliminar audio
- Muestra el ahorro en MB al finalizar

## 🚀 Uso
Abre `index.html` en cualquier navegador moderno.
No necesita servidor ni instalación.

## ⚠️ Nota
La primera compresión descarga el core de FFmpeg (~30MB).
